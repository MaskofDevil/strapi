# Strapi Blog

A Simple Blog created with strapi in the backend. Strapi Admin is configured with graphql to optimise the queries.

## Features

- Minimal
- CRUD Operations
- Backend Admin Page

## Tech Stack

**Client:** HTML, CSS, React, react-router-dom

**Server:** strapi, graphql, sqlite3
